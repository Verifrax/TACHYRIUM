# Code of Conduct

## Purpose

TACHYRIUM is maintained as a bounded cognition repository inside the Verifrax stack.

## Expected conduct

Contributors and operators should:

- keep changes bounded and reviewable
- preserve repository role clarity
- avoid sovereign role drift
- document boundary implications when proposing changes
- keep discussion precise, technical, and non-destructive

## Unacceptable conduct

- proposing sovereign role capture inside TACHYRIUM
- obscuring boundary changes
- misrepresenting subordinate outputs as authoritative truth
- hostile or destructive collaboration behavior

## Repository principle

Technical and social conduct both must preserve legibility, reviewability, and non-sovereign operation.
