# TACHYRIUM Boundary Manifest

## Purpose

This manifest states the hard repository boundary in one place.

## Primary role

TACHYRIUM is sovereign cognition infrastructure for the Verifrax stack.

## Allowed functions

* bounded candidate generation
* bounded routing
* bounded classification
* bounded topology mapping
* bounded contradiction analysis
* bounded briefing generation
* bounded cognition pipelines
* subordinate operator-facing summaries
* unified bounded CLI dispatch
* shared bounded object construction

## Forbidden functions

* constitutional law
* accepted world-state authorship
* authority-of-record issuance
* governed execution of record
* final merge enforcement
* final verification verdicts
* proof publication truth
* primitive truth authorship

## Boundary rule

Every executable surface in this repository must remain subordinate, reviewable, and non-sovereign.

## Current bounded executable surfaces

* `app/object_builder.py`
* `app/tachyrium_cli.py`
* `app/generate_candidate_object.py`
* `app/route_candidate_target.py`
* `app/classify_subject.py`
* `app/map_topology.py`
* `app/detect_contradiction.py`
* `app/build_briefing.py`
* `app/run_pipeline.py`

## Enforcement principle

If a future surface would require sovereign truth, authority, execution legitimacy, or final verification judgment, it is out of scope for TACHYRIUM.
