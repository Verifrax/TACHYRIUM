# TACHYRIUM Prompt Contract

## Purpose

This document defines the lawful prompt boundary for TACHYRIUM.

It specifies what a prompt may ask TACHYRIUM to do, what a prompt must provide for bounded operation, and which prompt patterns are forbidden because they would push TACHYRIUM across sovereign or adjacent Verifrax boundaries.

## Prompt axiom

```text
prompt in -> bounded cognition out
```

A valid prompt must preserve:

* explicit scope
* explicit target
* explicit reviewability
* subordinate output status

## Required prompt properties

A lawful TACHYRIUM prompt should identify:

* the target surface or object
* the task type
* the intended output class
* any relevant boundary constraints
* whether the result is draft, summary, suggestion, or analysis

## Allowed prompt intents

Allowed prompt intents include:

* clarify repo role
* summarize topology
* diagnose contradiction
* propose bounded routing
* draft subordinate README text
* draft subordinate docs text
* draft subordinate host copy
* compress evidence into operator-brief form
* transform ambiguity into reviewable candidate structure

## Forbidden prompt intents

Forbidden prompt intents include requests to:

* define constitutional law
* mutate accepted epoch truth
* issue authority
* authorize execution
* emit authoritative receipts
* block or allow merges as final gate
* determine final verification verdict
* replace proof publication truth
* impersonate any sovereign repo primary role

## Prompt typing rule

Each prompt should map cleanly to one of these output intentions:

* candidate
* draft
* summary
* suggestion
* briefing
* analysis

If the requested result cannot be typed this way, the prompt is too strong for lawful TACHYRIUM operation.

## Minimum bounded prompt form

A minimally lawful prompt should make these fields obvious:

* subject
* target
* constraint
* desired output type

## Stability rule

Prompts must remain narrower than the repository mission, the interface contract, and the output contract.

If a prompt attempts to pull TACHYRIUM into sovereign truth, authority, execution, enforcement, or final verification judgment, that prompt is outside the current lawful scope of the repo.
