# Contributing to TACHYRIUM

## Purpose

This repository accepts contributions only when they preserve the bounded cognition role of TACHYRIUM.

## Contribution rule

Every change must remain:

- subordinate
- reviewable
- non-sovereign
- aligned with the boundary manifest

## Allowed contribution classes

- doctrine clarification
- contract refinement
- schema refinement
- validation improvements
- smoke tests
- bounded cognition seeds
- operator documentation
- repository inventory and release legibility

## Forbidden contribution classes

- law authorship
- accepted state authorship
- authority issuance
- governed execution of record
- final merge enforcement
- final verification verdict logic
- proof truth publication

## Review expectation

Contributors must verify that any new executable surface remains bounded and does not cross into sovereign role capture.
